Общая продолжительность работы:
Выполнено за 6-7 часов.

Тестирование:
Реализованные функциональности протестированы в браузере Google Chrome.

Используемые технологии:
Код написан на ванильном JavaScript, мог использовать Bootstrap и jQuery для облегчения работы, но решил что просто на JS будет интереснее.

- News:
При нажатии на кнопку Explore, открывается модальное окно с отображением полной версии выбранной новости.
Верхние новости интегрированы с главным блоком. При выборе конкретной новости из верхнего списка, она автоматически отображается с изображением на главной странице.
Каждые 10 секунд главная новость обновляется из предопределенного массива новостей.
Реализована загрузка дополнительных новостей, если количество последних новостей превышает 8.
Разработана и протестирована адаптивная верстка для корректного отображения на разных устройствах.

- Chart:
Использован плагин Chart.js для отображения интерактивных графиков.
Реализована навигация по датам в виде карусели. При выборе даты отображается случайное значение объема данных (в формате GB) для текущего периода.
Данные для графика генерируются случайным образом, обеспечивая динамическое поведение.
Настроена адаптивная верстка для корректного отображения графиков на устройствах с различными разрешениями.
